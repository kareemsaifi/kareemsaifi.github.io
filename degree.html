I study Artificial Intelligence and Computer Science at the University of Sheffield

My Degree

In the department of computer science, they teach us several modules in our first year to get us ready to pursue a career in our degree

Web & Internet Technology

In this module we are taught how to create websites using HTML, CSS and Javascript. We have worked on a data visualisation canvas with javascript, make sure to check it out if you have the time

Java Programming

In this module we learn object oriented programming using the java programming language. We have done various projects that have to do with java from basic projects reaching some object oriented projects

Machines & Intelligence

This module is all about building intelligent machines, the history of Artificial intelligence and some working with and online simulation of the Miro Bot. In later stages of the module we started learning about how we see things and how to implement this into computer vision

Introduction to Software Engineering

With the teaching of basic software engineering concepts to create programs that meet specification, along with working in a team and an introduction to programming using the Ruby programming language and using github for version control this module prepares the students in the Department of computer science to important skills needed for this field

Foundations in Computer Science

The field of computer science relies heavily on mathematical concepts relating to logic, proof, functions and relations and this module provides students with what they need to be able to progress into future computing courses

Cognitive Psychology

Since I am doing artificial intelligence with Computer Science the field of psychology and specially cognitive psychology is very essential, understanding how our mind works and how we see/feel things, how we come up with decisions and other things.
